
# Anonymous Chat WhatsApp

Can help you send WhatsApp messages without having to save the number 


## Usage

Clone the project

```bash
  git clone https://github.com/rakaatmaja/anonymchat
```

Go to the project directory

```bash
  cd chat_anonym
```

Get package in terminal

```bash
  flutter pub get
```

Run the project

```bash
  flutter run
```
